trim_small_clustersÂ¶

an example for the use of porespy.filters.trim_small_clusters

[1]:

import numpy as np
import porespy as ps
import scipy.ndimage as spim
import matplotlib.pyplot as plt
import skimage

trim_small_clusters removes clusters from images if they are smaller than the size given as an input.

Generate input images:

[2]:

im = np.zeros([300, 300])
im1 = ps.generators.RSA(im, radius=5, volume_fraction=0.5)
im2 = ps.generators.RSA(im, radius=4, volume_fraction=0.5)
im3 = ps.generators.RSA(im, radius=3, volume_fraction=0.5)
im4 = ps.generators.RSA(im, radius=2, volume_fraction=0.5)
im5 = ps.generators.RSA(im, radius=1, volume_fraction=0.5)


fig, ax = plt.subplots(2, 3, figsize=[15, 10]);
ax[0,0].imshow(im1);
ax[0,1].imshow(im2);
ax[0,2].imshow(im3);
ax[1,0].imshow(im4);
ax[1,1].imshow(im5);

ax[0,0].axis(False);
ax[0,1].axis(False);
ax[0,2].axis(False);
ax[1,0].axis(False);
ax[1,1].axis(False);
ax[1,2].axis(False);
